Parallel Agent Scheduler

Kubernetes Pod-Based Agent Orchestration — Claude Agent Team Demo

Architecture: Agents run as isolated Kubernetes pods instead of tmux sessions. In Swarm Mode, an uber-agent decomposes your idea and spawns parallel worker pods. In Team Mode, a delegate-mode lead coordinates role-specialized agents (architect, K8s expert, devil's advocate, planner) — each running in their own pod. The lead is restricted to coordination-only actions: spawning, messaging, shutting down agents, and managing tasks.
